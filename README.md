# Movie Recommendation System

## Overview
This project is a collaborative filtering based movie recommendation system built using MovieLens datasets.

The system recommends similar movies based on user rating behavior using cosine similarity.

---

## Features

- Data preprocessing
- Collaborative filtering
- Cosine similarity
- Recommendation generation
- Model saving
- Visualizations
- Professional ML pipeline structure

---

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
- Joblib
